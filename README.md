# TableCRM-testTask

#### Тестовое задание на позицию Frontend-разработчик в компанию "TableCRM":

Описание: Сделать мобильную форму оформления заказа в формате webapp для tablecrm.com.

<br>

По данной <a href='https://app.tablecrm.com/docs_sales?token=af1874616430e04cfd4bce30035789907e899fc7c3a1a4bb27254828ff304a77'>ссылке</a> можно вызвать модальное окно создания продажи и увидеть основные ее компоненты.

**Мобильная форма создания заказа**:

- Форма для ввода токена для авторизации в конкретной кассе.
- Ввод телефона клиента и поиск по телефону
- Вывод всех счетов как в модальном окне оформления продажи.
- Вывод всех организаций как в модальном окне оформления продажи.
- Вывод всех складов как в модальном окне оформления продажи.
- Вывод типов цен как в модальном окне оформления продажи.
- Выбор товаров как в модальном окне оформления продажи.
- Кнопки создать продажи и создать и провести.

<br>
